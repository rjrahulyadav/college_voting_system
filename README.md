# College Voting System

A modern, secure digital voting system built for SONA College of Technology with beautiful neon UI design.

## Features

- **Modern Neon UI Design** - Beautiful cyan and magenta themed interface
- **Secure Authentication** - Separate login portals for voters and administrators
- **Multi-Year Support** - Supports 1st, 2nd, 3rd, and 4th year students
- **Real-time Vote Counting** - Instant vote tallying and results
- **Admin Dashboard** - Complete management system for candidates and voters
- **Responsive Design** - Works on desktop, tablet, and mobile devices
- **Indian Student Database** - Pre-populated with Indian names and data

## Technology Stack

- **Frontend**: HTML5, CSS3, JavaScript
- **Backend**: PHP
- **Database**: MySQL
- **Styling**: Custom CSS with neon effects, Google Fonts (Orbitron, Rajdhani)
- **Server**: Apache (XAMPP)

## Installation

1. Install XAMPP server
2. Clone this repository to `htdocs` folder
3. Import the database from `db/ovs_indian_names.sql`
4. Configure database connection in `dbcon.php`
5. Access the system via `localhost/Online_voting_system`
